🛠️ How does it all work?

🖱️ Hover Mode

Enable this mode to translate text just by moving your mouse. Simply hover over the original text on the left side, and the script will automatically translate it in the background.

Tip: The script ignores already verified items (✅) and those previously translated by AI (✨). This prevents wasting your API limits.

⏭️ Auto-Skip Page

This option works together with the Auto-Translate mode. If enabled, the editor will automatically jump to the next page and continue translating once the current page is fully processed. Requires leaving the browser tab open.

🚀 Auto-Translate

Turn this switch ON to automatically translate items on the current page. It stays ON until you turn it off. If "Auto-Skip Page" is disabled, it will pause at the end of the page and wait for you to manually switch to the next one before resuming!

Tip: A safe delay is kept between each translation request (1.5s for Google Free API) to prevent your IP address from getting temporarily banned by Google.

🐛 Debug Mode (dv.json)

Downloads a special language file where every translation is replaced with its ID and English text (e.g., [42] Welcome). Upload this to your live server to easily find out which line corresponds to which translation. Search by this #ID in the panel!

📥 Download & Sync

Downloads the resulting {lang}.json file to your computer. It also synchronizes your old translations with the new ones, attempting to repair syntax errors.

🗑️ Reset Editor

Clears all your browser's local storage data (uploaded files, translations, toggles, memory) and gives you a fresh start. Unsaved progress will be lost!

🌐 Google Free (Public Extensions API)

1. Setup & Cost:

None required. Works out of the box and is 100% Free. No credit card required.

2. Limits & Performance:

Bound to your personal IP address. Auto-Translate is artificially slowed down (~1.5s per request) to prevent your IP from getting a temporary Google ban.

☁️ Google Cloud Translation API Guide

1. Pricing & Limits:

First

500,000 characters/month are free

You can translate up to 6,000,000 chars per minute. A credit card is required by Google to create the API Key to prevent spam, but you will not be charged if you stay under the limit.

2. How to get the key:

  • Go to the Google Cloud Console ↗ and create a new project.
  • Enable Billing (Navigation Menu > Billing).
  • Go to Cloud Translation API ↗ and click Enable.
  • Go to Navigation Menu > APIs & Services > Credentials.
  • Click Create Credentials > API key.
  • Important Security Step: Edit the key, scroll down to API restrictions, choose Restrict key and select exactly Cloud Translation API. Leave Application restrictions at "None".
  • Save and paste the key below.
